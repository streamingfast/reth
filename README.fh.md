## Reth Firehose Fork

This is a fork of Reth with Firehose tracing built in directly within the node execution code runtime. You branching model is:

- [firehose/1.x](https://github.com/streamingfast/reth/tree/firehose/1.x) for Reth 1.x (phasing out).
- [firehose/2.x](https://github.com/streamingfast/reth/tree/firehose/2.x) for Reth 2.x (phasing out).
